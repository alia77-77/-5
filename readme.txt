Есть два интерфейса SomeInterface и SomeOtherInterface и по 2 реализации для каждого интерфейса. 
Они  выводят разные строки в консоль при вызове методов.
В файле app.properties указывается интерфейс и реализация, которую нужно применить при создании класса, реализующего интерфейс. 
AutoInjectable это аннотация, которую нужно сделать исходя из задания.
ElementType.Field в аннотации для того, чтобы ее можно было применять на полях класса.
RetentionPolicy.Runtime для того, чтобы была видна во время выполнения программы. 
SomeBean - класс, в котором два поля с аннотацией и метод для вызова методов тех двух объектов.
Класс Injector заполняет поля переданного ему класса, объектами, которые указаны в app.properties. 
В констркукторе создается объект класса Properties. 
Метод inject принимает объект, в котором нужно создать объекты, и возвращает его же. 
Метод injectObject получает класс переданного объекта, затем получает все поля этого класса. 
Если поле имеет аннотацию, то пытаемся создать объект. 
Если объект создан успешно, то устанавливаем этому полю через setAccessible(true), что в это поле можно поместить объект из вне и затем устанавливаем объект.
 Метод createObject принимает интерфейс, объект которого нужно создать, затем пытается в properties найти этот интерфейс. 
Если находит, то создает объект класса, который указан против интерфейса, записанного в файле
